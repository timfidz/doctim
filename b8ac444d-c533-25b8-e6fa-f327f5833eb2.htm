Accecia.Clarisse.Services.Public.Contracts Namespace

This document describes the interfaces available for 3rd party users who wants to use Clarisse API in their softwares.

[This is preliminary documentation and is subject to change.]

This Namespace holds the interfaces exposing the Clarisse public API
Interfaces
  InterfaceDescription
IAdminService
The IAdminService interface provides methods for administration purpose. All of these methods needs the calling user to have Administration rights in order to be executed.
IFileService
The IFileService interface provides methods for file upload and download purpose.
IPollService
The IPollService interface provides methods to track status of ongoing long tasks that have already been launched.
ISearchService
The ISearchService interface provides methods related to search features.
ISecurityService
The ISecurityService interface provides methods related to license management and system version.
ISharedService
ISharedServiceChannel
The ISharedServiceChannel interface provides methods related to sharing operations.
ISystemService
The ISystemService interface provides methods related to software settings and mailing operations.
IUserService
The IUserService interface provides methods related to users, flows, tasks and task notes.
