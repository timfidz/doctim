Accecia.Clarisse.Services.Base.Contracts Namespace

This document describes the interfaces available for 3rd party users who wants to use Clarisse API in their softwares.

[This is preliminary documentation and is subject to change.]

This Namespace holds the interfaces implenting Clarisse public API
Interfaces
  InterfaceDescription
IBaseAdminService
The IBaseAdminService interface provides methods for administration purpose. All of these methods needs the calling user to have Administration rights in order to be executed.
IBaseFileService
The IBaseFileService interface provides methods for file upload and download purpose.
IBasePollService
The IBasePollService interface provides methods to track status of ongoing long tasks that have already been launched.
IBaseSearchService
The IBaseSearchService interface provides methods related to search features.
IBaseSecurityService
The IBaseSecurityService interface provides methods related to license management and system version.
IBaseSystemService
The IBaseSystemService interface provides methods related to software settings and mailing operations.
IBaseUserService
The IBaseUserService interface provides methods related to users, flows, tasks and task notes.
